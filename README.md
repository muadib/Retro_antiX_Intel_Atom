# Retro_antiX_Intel_Atom
Scripts para correr sobre una distro antiX ya instalada en una netbook Intel Atom.

La idea es preparar una netbook Intel Atom (Samsung N150 Plus) para poder emular juegos SNES, NES, etc. con Retro Arch. Y que además sea parecida a Batocera.
Batocera tiene una .iso especifica para Atom, pero es una versión vieja.

Este script instala:
- ES‑DE (AppImage)
- RetroArch (AppImage)
- Cores livianos
- PCSX‑Reloaded
- Mupen64Plus
- FS‑UAE
- DOSBox‑Staging
- Estructura de carpetas estilo Batocera
- Lanzadores
- Configuración base
